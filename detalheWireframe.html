Protótipo

Com base nos wireframes, desenvolvi um protótipo para o portfólio online. Esse protótipo permite visualizar o fluxo das páginas e comentários sobre o projeto. Ele foi essencial para identificar melhorias no design e garantir uma interface agradável ao usuário final.

Wireframes

Os wireframes de baixa fidelidade foram criados para esboçar a estrutura inicial do portfólio. Eles focam na organização dos elementos visuais e no layout das páginas, sem detalhamento estético. Essa etapa foi fundamental para definir a base do design e planejar a navegação antes de avançar para o protótipo.
